# Bask Health Application Checklist

## Portfolio Readiness

- Open Source Impact section includes Cloudflare, npmx, and iii-hq contributions.
- Hero positioning emphasizes TypeScript, Next.js, distributed systems, and 0-to-1 execution.
- Expertise section highlights systems architecture, cloud, and DevOps capabilities.
- Skills section maps to startup stack requirements (Next.js, SST, serverless, infra).

## Message of Interest

Use a short message that references their stack and your direct relevance:

"I am a Full-Stack Software Engineer focused on TypeScript/Next.js systems and distributed architecture. I have contributed to open-source projects with production constraints, including Cloudflare-related work and a11y-heavy platform refactors. Your SST + Next.js stack aligns directly with how I build and ship product systems." 

## Proof to Send

- Include one direct PR/commit link as your strongest technical proof.
- If needed, include one fallback link to your contribution search query.

## Submission Pack

- Portfolio URL
- GitHub profile URL
- 1 direct PR link
- 2-line personalized interest message
